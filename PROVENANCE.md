# Provenance

This repository is a clean public distribution of the Intrinsical Policy Engine
core.

It is derived from prior private development work, but it intentionally excludes
private Git history, private framework packs, policy calibrations, commercial
templates, sector demos, generated outputs, and reserved assets.

The public release boundary is intentional: this repository demonstrates the
engine and the pack-authoring model without publishing private regulatory or
commercial content.
