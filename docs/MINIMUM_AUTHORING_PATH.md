# Minimum Authoring Path

A useful pack can be very small.

The starter pack demonstrates this minimum path:

1. Define a few flags.
2. Add a few questions that emit those flags.
3. Add one or more rules that select actions.
4. Map actions to evidence templates.
5. Add a simple export profile.
6. Validate, export, and seal.

For a first pack, keep the surface intentionally narrow:

- one policy file or equivalent rule area;
- one question mapping;
- zero or one action family;
- one smoke test;
- one example answer file.

Avoid adding broad coverage claims until the pack has human review and real
acceptance criteria.
